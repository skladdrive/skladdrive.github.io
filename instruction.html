СкладDrive

Учет запчастей и склада для СТО

СкладDrive — portable-программа для Windows, которая помогает вести учет автозапчастей, складских остатков и движения товаров на СТО, в автомагазине или на небольшом складе.

Возможности программы

  • ✅ Учет остатков и движения товаров
  • ✅ Поступление и списание по штрихкоду или артикулу
  • ✅ Инвентаризация склада со сканером штрихкодов
  • ✅ Справочник запчастей и штрихкодов
  • ✅ Загрузка и обработка прайсов поставщиков
  • ✅ Объединение нескольких прайсов в один файл
  • ✅ Экспорт в Excel и CSV
  • ✅ Резервное копирование данных
  • ✅ Portable-версия без установки

Быстрый старт за 5 минут

1. Распакуйте архив

Распакуйте архив программы в отдельную папку на локальном диске.

Не запускайте программу прямо из ZIP-архива. Сначала распакуйте архив, затем откройте папку программы.

Для небольшой базы можно использовать папку, которая синхронизируется с облаком. Для большой базы лучше держать рабочую папку на локальном SSD/NVMe, а в облако копировать только резервные копии.

2. Запустите программу

Запустите файл SkladDrive.exe.

Установка не требуется — программа работает в portable-режиме.

При необходимости создайте ярлык на рабочем столе:

  1. Нажмите правой кнопкой мыши на файле SkladDrive.exe.
  2. Выберите «Создать ярлык».
  3. Переместите ярлык на рабочий стол.

3. Загрузите справочник запчастей

Для быстрого старта можно использовать один из подготовленных справочников:

ФайлСодержимое
spravochnik_EAN_186.xlsxОколо 186 000 позиций со штрихкодами
spravochnik_2M_EAN_186.xlsxБолее 2 млн позиций и 186 000 штрихкодов

Выберите подходящий вариант и загрузите его через раздел «Справочник».

4. Начните работу

Если сканера штрихкодов нет — используйте поиск по артикулу.

Постепенно добавляйте новые позиции в базу данных: описание, бренд, артикул, штрихкод, закупочную и продажную цену.

5. Используйте сканер штрихкодов

После наполнения базы рекомендуется использовать сканер штрихкодов.

Сканирование позволяет:

  • быстро находить нужные позиции;
  • принимать товар на склад;
  • списывать детали и запчасти;
  • ускорять проведение операций;
  • уменьшать количество ошибок при ручном вводе.
Рекомендация: настройте сканер в режиме клавиатуры USB HID / Keyboard Wedge, укажите префикс F7 и суффикс Enter. В СкладDrive клавиша F7 автоматически переводит курсор в строку поиска, поэтому перед сканированием не нужно каждый раз щёлкать мышью по полю ввода.

Настройка префикса и суффикса зависит от модели сканера. Обычно она выполняется через инструкцию производителя или сканированием служебных штрихкодов из руководства к устройству.

Если штрихкод отсутствует, найдите товар по артикулу и при необходимости добавьте штрихкод в карточку товара.

6. Создавайте резервные копии

Регулярно выполняйте резервное копирование данных через вкладку «Сервис».

Это поможет сохранить:

  • остатки на складе;
  • журнал операций;
  • справочник запчастей;
  • настройки программы;
  • отчеты и другие важные данные.

Перед крупным импортом, обновлением программы или экспериментами с прайсами обязательно сделайте резервную копию.

Лицензия

Без лицензии программа работает в пробном режиме 30 дней.

Для получения лицензии:

  1. Откройте вкладку «О приложении».
  2. Нажмите «Скопировать код».
  3. Отправьте код ПК на skladdrive@mail.ru.
  4. Полученный файл license.dat положите рядом с SkladDrive.exe.
  5. Перезапустите программу или нажмите «Проверить license.dat».
