# Сайт СкладDrive для GitHub Pages

Это готовая статическая страница для сервиса GitHub Pages. Ее можно открыть на своем компьютере без загрузки в GitHub, проверить текст и картинки, а потом залить файлы в репозиторий.

## Что внутри

- `index.html` - главная страница сайта.
- `styles.css` - оформление страницы.
- `gallery.js` - карусель со скриншотами.
- `preview.bat` - быстрый запуск локального предпросмотра.
- `assets/hero.png` - большой баннер.
- `assets/icon.png` - иконка приложения и favicon.
- `assets/screenshots/` - скриншоты интерфейса для галереи.
- `downloads/` - папка для файлов, которые посетители будут скачивать.
- `downloads/instruction.txt` - краткая инструкция для пользователей.
- `downloads/changelog.txt` - история изменений.
- `downloads/README.txt` - подсказка, какие файлы можно положить в папку загрузок.

## Как посмотреть сайт на компьютере

1. Откройте папку с сайтом.
2. Дважды нажмите на файл `preview.bat`.
3. Страница откроется в браузере.

Можно и без `preview.bat`: дважды нажмите на `index.html`.

Если картинки отображаются и кнопки ведут на нужные разделы, сайт готов к публикации.

## Что положить в папку downloads

Перед публикацией добавьте в папку `downloads` архив программы:

- `SkladDrive.zip` - архив программы.

Файлы `instruction.txt` и `changelog.txt` уже созданы. Их можно редактировать обычным Блокнотом.

## Как загрузить сайт на GitHub Pages

1. Зайдите на сайт GitHub и создайте новый репозиторий.
2. Название можно сделать, например: `skladdrive-site`.
3. Откройте созданный репозиторий.
4. Нажмите кнопку `Add file`, затем `Upload files`.
5. Перетащите туда все файлы и папки из этой папки сайта:
   - `index.html`
   - `styles.css`
   - `gallery.js`
   - `preview.bat`
   - `assets`
   - `downloads`
   - `README.md`
6. Внизу страницы нажмите зеленую кнопку `Commit changes`.
7. Откройте настройки репозитория: `Settings`.
8. В левом меню найдите `Pages`.
9. В блоке `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - папка: `/root`
10. Нажмите `Save`.

Через 1-3 минуты GitHub покажет ссылку на сайт. Обычно она выглядит примерно так:

`https://ваш-логин.github.io/skladdrive-site/`

## Как обновлять сайт

Для замены программы загрузите новый архив в папку `downloads` с тем же именем `SkladDrive.zip`. Потом снова загрузите измененный файл на GitHub через `Add file` и `Upload files`.

Для изменения текста откройте `index.html` в любом текстовом редакторе, измените нужные строки и загрузите файл на GitHub заново.

Важно: в архив программы желательно положить подготовленный файл `spravochnik.xlsx`, чтобы пользователь мог быстро импортировать стартовый справочник в программе.
